Database Diagram Tool

Connect to any MySQL, PostgreSQL, or SQL Server database — or upload a SQL dump — to generate interactive diagrams

MySQL / MariaDB — How to connect

Connection details
  • Default port: 3306
  • Default user: root
  • Find your host in your hosting panel, or use localhost for local installs
Export a SQL dump (alternative)

Using MySQL Workbench: Server → Data Export → select schema → Export to Self-Contained File

Using the command line:

mysqldump -u root -p --no-data my_database > dump.sql

PostgreSQL — How to connect

Connection details
  • Default port: 5432
  • Default user: postgres
  • Find your host in your cloud dashboard, or use localhost for local installs
Export a SQL dump (alternative)

Using pgAdmin: Right-click database → Backup → Format: Plain → Only schema

Using the command line:

pg_dump -U postgres --schema-only my_database > dump.sql

SQL Server — How to connect

Connection details
  • Default port: 1433
  • Default user: sa (SQL auth) or leave blank for Windows Authentication
  • For Azure SQL, use your-server.database.windows.net as host
Export with SQL Server Management Studio (SSMS)

Right-click database → Tasks → Generate Scripts → Script entire database → choose "Schema Only" → Save to file

Using the command line (sqlcmd):
sqlcmd -S localhost -d my_database -E -Q "SET NOCOUNT ON" -i schema_script.sql -o dump.sql

Or use mssql-scripter:

mssql-scripter -S localhost -d my_database --schema-only -f dump.sql
